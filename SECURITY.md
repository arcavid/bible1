# Security policy

Please report security issues privately to the maintainer instead of opening a public issue with exploit details.

## Secrets

Never commit `.env.local`, Supabase service-role keys, direct database URLs, corpus seed files, or generated SQL containing protected corpus data.

## Corpus rights

Do not submit pull requests that add full Bible translation text unless the translation rights, source/provenance, and attribution requirements are explicitly documented and accepted by the maintainer.
